🎓 Crypto-Class - מערכת ניהול נוכחות מבוססת טוקנים
📖 תיאור הפרויקט
Crypto-Class היא מערכת ניהול נוכחות ומשימות מבוססת טוקנים (בלוקצ'יין) לכיתות לימוד. המערכת משלבת בוט טלגרם עם אתר אינטרנט מלא לניהול כיתה דיגיטלית מתקדמת.

✨ תכונות מרכזיות:
🤖 בוט טלגרם מלא עם פקודות אינטראקטיביות

📊 אתר אינטרנט עם דשבורד מורים וסטטיסטיקות

🏆 מערכת טוקנים לתגמול תלמידים

👥 מערכת הפניות (חבר מביא חבר)

📈 טבלת מובילים וסטטיסטיקות בזמן אמת

🔐 דשבורד מורים מאובטח עם הרשאות

💾 מסד נתונים מלא עם SQLAlchemy
